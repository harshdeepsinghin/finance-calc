The Real Rate of Return

Formula & Mathematical Approach

Computes the actual return of an investment after stripping out the impact of inflation and taxes. 

Post-Tax Return = Nominal Return * (1 - Tax Rate / 100)
Real Return = (1 + Post-Tax Return / 100) / (1 + Inflation / 100) - 1

Key Assumptions

  • Fisher equation is used for real return.
  • Tax rate is applied flatly to the nominal rate.
